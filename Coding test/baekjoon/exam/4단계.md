### **2562**
최댓값
여러 줄 입력 받기
```
s_list = [int(input()) for _ in range(N)]
```
N은 줄 숫자를 이야기한다.

위치 반환(index)
index(x) 함수는 리스트에 x 값이 있으면 x의 위치 값을 돌려준다.
```
print(s_list.index(max(s_list)) + 1)
```
