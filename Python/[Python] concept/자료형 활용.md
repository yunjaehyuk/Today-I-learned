### 5-1.리스트

```python
subway = ["유재석", "하하", "정형돈"]
print(subway)
#사람 추가하기
subway.append("조세호")
print(subway)
#사람을 위치 사이에 넣기
subway.insert(1,"박명수")
print(subway)
subway.pop()
subway.pop()
subway.pop()
print(subway)

#같은 이름의 사람이 몇 명 있는지 확인
subway.append("유재석")
print(subway.count("유재석"))

#정렬하기
num_list = [5,4,3,2,1]
print(num_list)
num_list.sort()
print(num_list)

#순서 뒤집기
num_list.reverse()
print(num_list)

#모두 지우기
num_list.clear()
print(num_list)

#리스트 확장
mix_list = [20, true, "조세호"]
num_list.extend(mix_list)
print(num_list)
```

### 5-2.사전

대응관계를 나타내는 자료형

```python

cabinet = {5:"유재석", 100:"김태호"}
print(cabinet)

print(cabinet.get(5))
print(cabinet.get(100))
print(cabinet[100])

# 딕셔너리에 내 키가 있는지 확인하기
print(5 in cabinet) #True
print(12 in cabinet) #False

#캐비넷 키 정수형
cabinet = {"A-3" : "유재석", "B-10" : "김종국"}
print(cabinet)
cabinet["A-3"] = "박명수"
print(cabinet)

#새 손님
cabinet["A-5"] = "정형돈"
print(cabinet)

#간 손님
del cabinet["A-3"]
print(cabinet)

#key들만 출력
print(cabinet.keys())

#value만 출력
print(cabinet.values())

#key,value 쌍으로 출력
print(cabinet.items())

#목욕탕 폐정
cabinet.clear()
print(cabinet)
```

### 5-3.튜플

리스트는 []으로 둘러싸지만 튜플은 ()으로 둘러싸인다. 리스트는  그 값의 생성, 삭제, 수정이 가능하지만 튜플은 그 값을 바꿀 수 없다.

```python
menu = ("돈가스" , "떡볶이")
print(menu)

name = "김종국"
age = 20
hobby = "코딩"

print(name, age, hobby)
(name, age, hobby) = ("김종국", 20, "코딩")
print(name, age, hobby)
```

사용이유 

값을 변화시킬려면 리스트를 아니면 튜플을 사용한다.

### 5-4.세트

```python
# 집합(set)
#중복 안됨, 순서 없음

my_set = {1,2,3,4,5,5,5}
print(my_set)

java = {"유재석", "정형돈", "박명수"}
python = set(["유재석", "박명수"])

# 교집합(java와 python을 모두할 수 있는 개발자)
print(java & python)

#합집합(java도 할 수 있고, python도 할 수 있는 개발자)
print(java | python)

#차집합(java는 할 수 있지만 python은 할 줄 모르는 개발자)
print(java - python)

#python할 줄 아는 사람이 늘어남
python.add("김태호")
print(python)

#python 할 줄 모르게 됨
python.remove("김태호")
print(python)
```
