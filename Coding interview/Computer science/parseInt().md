## 문자열 숫자로 변환하기parseInt()

### parseInt()

```jsx
parseInt(string, radix); 
```

**string**  

 - 숫자로 변환할 문자열

**radix**   

- optional   

- string 문자열을 읽을 진법(수의 진법 체계의 진법)  

 - 2~36의 수 리턴 값

string을 정수로 변환한 값을 리턴합니다.

만약, string의 첫 글자를 정수로 변경할 수 없으면NaN(Not a Number) 값을 리턴합니다.
