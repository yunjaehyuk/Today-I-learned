## Part 1.13 **변수 심화학습시간 & 저번시간 숙제**

변수의 선언, 할당, 범위

**변수의 범위**

```jsx
function 함수() {
 var 이름 = 'kim';
 console.log(이름);
}
함수안에서 변수 만들면 사용가능 범위는 함수 내부
```

**var, let, const 차이**

재선언, 재할당, 범위

var 재선언 O 재할당 O 범위 function

let 재선언 X  재할당 O 범위 {}

나중에 변수 만드는 거 또 만드는 거를 막는다.

const 재선언 X 재할당 X 범위 {}

값을 수정하면 큰일이 나는 변수를 만들 떄 사용
